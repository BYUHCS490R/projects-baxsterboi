Internet

The internet is basically a worldwide network that connects devices such as computers or phones to help people communicate, access information and use social media by using specific standard communication protocols like TCP or IP.

Web

The web is a type of interlinked digital pages and information that can be accessed through internet browser apps such as google.

How are the Internet and Web different?

The Internet is global network that allows data to be sent from device to another device while web is something that runs on the internet. In short, the Internet is a foundation, and the web is built upon it.

ASCII and Unicode

ASCII(American standard code for information Interchange) is type of coding that represents text using numbers such English letters and basic control characters, using 7 bits(128 characters). Meanwhile, Unicode is type of coding that designed for languages worldwide, including emojis and using 100,000 characters.

What is UTF-8

UTF-8 is a standard coding system that converts Unicode characters into bytes. It supports all languages,symbols and emojis.

URL

URL (Uniform resource locator)is type of link that helps locate information on the internet.

Domain Name

A domain name is a human-readable address to idenfitify specific websites on the Internet. It translates into an IP address to make it easier to locate the server.

HTTP

HTTP is a protocol that used to help web browser and web server communicate and transfer data.

HTTPS

HTTPS is a protocol secure that helps protect your data when browsing the web.

IP address

IP Address is a numerical identifier to provide network for devices so it can send and receive digital data over the internet.

DNS

DNS is a system that interprets domain into IP adresses to locate any websites on the internet.

W3C Recommendations

W3C Recommendations are the standard that help retain the function of the web.

XML, XHTML, HTML, HTML5

XML is a markup language that focused on storing and transporting data. XHTML is a markup language is an advanced version of XML which required well-fromed code. HTML is a markup language to create web by using headings and paragraphs. Lastly, HTML 5 is a markup language that helps nowadays web applications that added miltimedia, semantic elements and better performance.
